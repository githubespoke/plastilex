About & Methodology

Data sources, acquisition protocol, analysis methods, and known limitations

Project Overview

PlastiLex is a student-led bibliometric project that bridges the Chinese–English language gap in microplastics/plastics contamination research. Rather than translating copyrighted papers, PlastiLex builds an open, interoperable metadata index and produces novel analytical products from that metadata.

This tool demonstrates the analytical scope achievable through metadata-only methods: trend analysis, collaboration mapping, institutional geography, and keyword co-occurrence — all without reproducing any copyrighted content.

Data Sources

  • Chinese corpus: CNKI (中国知网), Wanfang Data (万方数据), CQVIP
  • English corpus: Web of Science Core Collection, Scopus
  • Time span: 2015–2024 (inclusive)
  • Query terms: microplastics, nanoplastics, plastic pollution, aquaculture, marine contamination (+ Chinese equivalents)

Known Limitations

Users should be aware of the following constraints:

Coverage bias CNKI/Wanfang access limited to subscribed journals for this study period.

Language parsing Chinese keyword extraction relies on jieba tokenisation; compound terms may be split.

Author disambiguation Name deduplication for Chinese authors is approximate — romanisation variations may create duplicate nodes.

Citation data Cross-language citation links are incomplete; only DOI-resolvable references captured.

Mock data notice This mockup uses plausible synthetic data for demonstration; real data collection is in progress.

Acquisition Protocol

1
Query construction — Boolean search queries built in both languages with MeSH-equivalent controlled vocabulary. Validated against reference seed sets of known papers.
2
Metadata export — Title, author names, affiliations, abstract keywords, DOI, journal, year, language exported as structured records. No full-text downloaded.
3
Deduplication — Cross-database duplicates removed via DOI matching, then fuzzy title+author matching for records lacking DOIs.
4
Entity resolution — Author name disambiguation via institutional affiliation clustering + ORCID lookups where available.
5
Network construction — Co-authorship edges built from shared paper membership. Keyword co-occurrence edges built from shared abstract keyword appearance.
6
Open publication — Final metadata corpus deposited to Zenodo under CC BY 4.0. No copyrighted content reproduced.

Analysis Methods

  • Trend analysis: Annual publication counts per topic, computed from keyword-classified records using LDA topic modelling (k=8).
  • Network analysis: Co-authorship graph computed with NetworkX; force layout via D3.js ForceSimulation.
  • Geolocation: Institutional affiliations geocoded using OpenStreetMap Nominatim API; manual correction for top-50 institutions.
  • Keyword analysis: Chinese keywords extracted with jieba; English with NLTK. Co-occurrence matrix built with sliding window (w=1 sentence).
  • Language bridging: Cross-language author identification via institutional affiliation and self-reported language competency indicators in ORCID profiles.
